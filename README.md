# Computerized Cognitive Retraining Program for Children with Disabilities

A comprehensive computerized cognitive retraining program that combines EEG neuro-feedback with home training to help children with developmental disabilities improve their cognitive abilities.

## ğŸ¯ Project Overview

This project addresses the critical need for accessible, effective cognitive training for children with developmental disabilities. It combines:

- **EEG Neuro-Feedback Training** - Real-time brain activity monitoring
- **Home Training Modules** - Accessible cognitive exercises
- **Integrated Monitoring System** - Clinician dashboard for progress tracking
- **Single Software Platform** - Unified interface for both training and monitoring

## ğŸš€ Quick Start

### 1. Clone the Repository
```bash
git clone <repository-url>
cd cognitive-retraining-program
```

### 2. Install Dependencies

#### Backend Dependencies
```bash
npm install
```

#### Frontend Dependencies
```bash
cd client
npm install
cd ..
```

### 3. Environment Setup

Create a `.env` file in the root directory:
```env
# Server Configuration
PORT=5000
NODE_ENV=development

# Database
MONGODB_URI=mongodb://localhost:27017/cognitive-training

# JWT Secret
JWT_SECRET=your-super-secret-jwt-key

# Client URL
CLIENT_URL=http://localhost:3000

# Email Configuration (optional)
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-email-password
```

### 4. Database Setup

Start MongoDB:
```bash
# On macOS with Homebrew
brew services start mongodb-community

# On Ubuntu/Debian
sudo systemctl start mongod

# On Windows
net start MongoDB
```

### 5. Run the Application

#### Development Mode (with hot reload)
```bash
# Start both backend and frontend
npm run dev:full

# Or start them separately:
# Backend only
npm run dev

# Frontend only
npm run client
```

#### Production Mode
```bash
# Build frontend
npm run build

# Start production server
npm start
```

### 6. Access the Application

- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
- **Health Check**: http://localhost:5000/health

## ğŸ”§ API Endpoints

### Authentication
- `POST /api/auth/register` - User registration
- `POST /api/auth/login` - User login
- `POST /api/auth/logout` - User logout
- `POST /api/auth/refresh` - Refresh token

### Users
- `GET /api/users/profile` - Get user profile
- `PUT /api/users/profile` - Update user profile
- `GET /api/users/children` - Get user's children
- `GET /api/users/patients` - Get clinician's patients

### Training
- `POST /api/training/sessions` - Create training session
- `GET /api/training/sessions` - Get user's sessions
- `PUT /api/training/sessions/:id` - Update session
- `DELETE /api/training/sessions/:id` - Delete session

### EEG Data
- `POST /api/eeg/data` - Upload EEG data
- `GET /api/eeg/sessions/:id` - Get session EEG data
- `GET /api/eeg/analytics` - Get EEG analytics

### Progress
- `GET /api/progress/overview` - Get progress overview
- `GET /api/progress/charts` - Get progress charts
- `GET /api/progress/analytics` - Get detailed analytics
